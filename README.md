# Anotações de Aula

Este repositório foi criado para reunir minhas anotações de aula de forma organizada, facilitando os estudos, revisões e consultas ao longo do semestre.

## Objetivo

Guardar conteúdos estudados nas disciplinas da faculdade, incluindo:

- resumos
- exercícios
- exemplos vistos em aula
- observações importantes
- materiais de apoio
