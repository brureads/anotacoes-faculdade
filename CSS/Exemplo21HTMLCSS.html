Exemplo 21 HTML CSS - Formas de vinculção de arquivos CSS

Vinculação de arquivos

Quando falamos de vinculação de arquivos CSS, falamos basicamente de duas formas a vinculação interna e a vinculação externa. Neste exemplo, vamos falar da vinculação externa que é a mais recomendada no desenvolvimento web, pois mantém o contéudo HTML separado do CSS.

A vinculação externa é a mais recomendada pois além de manter o codigo mais organizado facilita a manutenção e permite que voce use o mesmo estilo em varias paginas do seu site, ou seja, permite o reuso.

Vinculação Externa CSS

Como vimos a cima, a vinculação a externa é a mais recomendada pois possibilita o reuso e facilita a manutencao das paginas centralizando tudo em um unico arquivo CSS. Para criarmos um arquivo .css, devemos seguir os seguintes passos:

  • Criar o arquivo com extensão .css - Neste arquivo, devemos escrever apenas as regras de estilo, usando os seletores que ja conhecemos(tipo, classe e id)
  • Vincular os arquivos css criado ao documento html, para isso usaremos a tag <link>

Vantagens da vinculação Externa

  • Cache do Navegador: o navegador baixa o CSS apenas uma vez e o armazena. Isso torna o carregamento das outras paginas do site muito mais rapido
  • Organização: como o codigo fica separado, nao ha poluição ou mistura de html com css
  • Edição Global: neste caso, estamos falando da facilidade de manutenção, pois se precisarmos mudar um estilo, basta fazer alteração no arquivo separado, que a mudança se reflitira em todos os elementos que utilizam aquela regra de estilo
