# Introdução à Rede de Computadores

## Conceito
- Rede de computadores é um **conjunto de dispositivos conectados** para permitir a **troca de informações**.
- O principal objetivo é **conectar dispositivos** e possibilitar comunicação **rápida e eficiente**.

## Importância das redes
- As redes tornaram o mundo mais interligado.
- Permitem acessar:
  - informações;
  - serviços;
  - pessoas;
  - recursos digitais de qualquer lugar.

## Cabos submarinos
- Os **cabos submarinos** são a base da comunicação global.
- Eles transportam a maior parte do tráfego de dados entre continentes.
- São feitos de **fibra óptica** e instalados no fundo do oceano.

## Comunicação e sociedade
- A comunicação é essencial para o desenvolvimento humano e social.
- Ela é importante para:
  - transmitir conhecimento;
  - criar relacionamentos;
  - cooperar e colaborar;
  - transmitir cultura e valores;
  - tomar decisões;
  - resolver conflitos;
  - promover inclusão social.

## Redes e comunicação
- As redes de computadores são fundamentais para a comunicação:
  - **pessoal**;
  - **governamental**;
  - **empresarial**.

### Comunicação pessoal
- Redes sociais;
- e-mails;
- mensagens instantâneas;
- chamadas de voz e vídeo por VoIP.

### Comunicação governamental
- Governo eletrônico;
- comunicação entre órgãos;
- divulgação de notícias e informações oficiais.

### Comunicação empresarial
- Redes corporativas;
- videoconferências;
- transferência de dados;
- comércio eletrônico.

## Benefícios das redes
- Os principais benefícios são:
  - **troca de informações**;
  - **compartilhamento de recursos**.

## Primeiras aplicações de rede
- As primeiras redes atendiam principalmente:
  - transferência de arquivos;
  - login remoto;
  - correio eletrônico;
  - fóruns (*newsgroups*).

## Requisitos de uma rede

### Desempenho
- Relacionado à velocidade da comunicação.
- Depende de:
  - **latência**;
  - **taxa de transmissão**.

### Confiabilidade
- Deve garantir resistência a falhas.
- Deve conseguir se recuperar de problemas.

### Segurança
- Protege contra acessos indevidos externos.
- Controla acesso a recursos da rede.

### Confidencialidade
- As informações só devem estar disponíveis para quem tem autorização.

### Mobilidade
- Permite reconexão de dispositivos móveis em diferentes pontos da rede.

### Qualidade de Serviço (QoS)
- Garante atendimento a prazos de transmissão e processamento.
- Exige:
  - largura de banda adequada;
  - latência controlada.

## Elementos da rede
- **Roteador**: encaminha pacotes entre redes diferentes.
- **Switch**: conecta dispositivos da mesma rede local e envia os dados ao destino correto.
- **Hub**: conecta dispositivos, mas envia os dados para todos os equipamentos ligados.
- **Modem**: conecta a rede local a uma rede externa, como a internet.

## Pacotes de dados
- Os dados trafegam na rede em forma de **pacotes**.
- Cada pacote possui:
  - **cabeçalho**;
  - **dados**;
  - **trailer**.

## Etapas do envio de pacotes
- **Fragmentação**: divide os dados em partes menores.
- **Encapsulamento**: adiciona cabeçalho e trailer.
- **Roteamento**: define o caminho até o destino.
- **Reassembly**: reorganiza os pacotes no destino.

## Enlace
- Enlace é a **conexão física ou lógica** entre dois dispositivos.
- Pode ocorrer por:
  - cabos;
  - fibra óptica;
  - conexão sem fio.

## Aspectos de projeto de rede
- O projeto de uma rede deve considerar:
  - hardware;
  - software;
  - heterogeneidade;
  - tecnologias de enlace;
  - topologia física;
  - abrangência geográfica.

## Heterogeneidade
- As redes podem usar tecnologias diferentes.
- Exemplo:
  - rede sem fio doméstica;
  - rede por fibra óptica.
- Para permitir interoperabilidade, usa-se uma **rede lógica**, como a internet.

## Tecnologias de enlace
- Podem ser:
  - **ponto a ponto**;
  - **multiponto**.
- Definem como os dados serão organizados e transmitidos.
- Exemplos:
  - Ethernet;
  - Wi-Fi;
  - Bluetooth;
  - Frame Relay;
  - Wi-Max.

## Topologias físicas
- **Estrela**: dispositivos ligados a um concentrador.
- **Anel**: os dados passam por todos os computadores do anel.
- **Barra**: dispositivos ligados a um meio compartilhado.
- **Malha/Grafo**: vários nós com ligações ponto a ponto.
- **Híbrida**: combinação de mais de uma topologia.

## Abrangência geográfica
- **PAN**: rede pessoal de curto alcance. Ex.: Bluetooth.
- **LAN**: rede local. Ex.: Ethernet e Wi-Fi.
- **MAN**: rede metropolitana. Ex.: Wi-Max.
- **WAN**: rede de longa distância, até mundial. Ex.: Frame Relay.
