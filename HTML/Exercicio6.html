Exemplo 9 - Links e Tabelas

Descrição

Este exemplo mostra como criar links e tabelas em HTML. As tags abordadas são:

  • <a>: Define um hyperlink.
  • <table>: Define uma tabela.
  • <tr>: Define uma linha da tabela.
  • <th>: Define um cabeçalho de tabela.
  • <td>: Define uma célula da tabela.

Exemplos Práticos
