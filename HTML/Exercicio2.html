Exemplo 5 HTML - Principais elementos HTML

Uso das tags <p>, < br>, < hr>, < strong>, < em>

Este exemplo demostra o uso de tags pra formatacao de texto e organização visual

Descricao das principais tags

  • < p>:Essa tag define um paragrafo.
  • < hr>:Esta tag cria uma linha horizontal no documento html.
  • < br>:Esta tag é utilizada na quebra de linha.
  • < strong>:Esta tag define que o elemento ficará em negrito e ela tem importancia semantica.
  • < em>:Esta tag define em italico o elemento e tambem tem enfase semantica.
