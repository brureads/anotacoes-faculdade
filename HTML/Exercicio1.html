Exemplo 4 HTML - Estrutura Geral

Este exemplo mostra a estrutura mínima de um documento HTML. A seguir são descritas as principais tags utilizadas nessa estrutura geral.

  • <DOCTYPE html>: É uma instrução html, essa tag define o tipo do documento html.
  • <html>: é o elemento central do documento, é container do documento html, ele engloba toda a página.
  • <head>: Essa tag contém os metadados que serão interpretados pelo navegador, tai como titulos e metatags de codificação.
  • <body>: Esse bloco é responsável or deixar visível o conteúdo destinado aos usuários. Ou seja todo o conteúdo do documento html.
