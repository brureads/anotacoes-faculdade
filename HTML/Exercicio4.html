Exemplo 7 - HTML Listas Ordenadas e Não Ordenadas

Este exemplo mostra como criar listas ordenadas e não ordenadas incluindo subitens(listas aninhadas).

Descrição das tags utilizadas na criação das listas.

  • <ol>:Define uma lista ordenada(com números, letras e ou algarismos romanos)
  • <ul>: Define uma lista não ordenada ( com marcadores com pontos e quadrados)
  • <li>: Define um elemento na lista

Exemplo Pratico

Listas Nao ordenadas - <ul>

  • Item 1 da lista
  • Item 2 da lista
    • subitem 2.1
    • subitem 2.2
  • Item 3 da lista
    • subitem 3.1
    • subitem 3.2
      • subitem 3.2.1
      • subitem 3.2.2
      • subitem 3.2.3
    • subitem 3.3
    • subitem 3.4
    • subitem 3.5
  • Item 4 da lista
  • Item 5 da lista

Listas Ordenadas - <ol>

  1. Item 1 da lista
  2. Item 2 da lista
    1. subitem 2.1
    2. subitem 2.2
  3. Item 3 da lista
    1. subitem 3.1
    2. subitem 3.2
      1. subitem 3.2.1
      2. subitem 3.2.2
      3. subitem 3.2.3
    3. subitem 3.3
    4. subitem 3.4
    5. subitem 3.5
  4. Item 4 da lista
    1. subitem 4.1
    2. subitem 4.2
    3. subitem 4.3
    4. subitem 4.4
  5. Item 5 da lista
