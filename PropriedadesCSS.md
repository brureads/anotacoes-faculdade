# PROPRIEDADES E VALORES
- Uma propriedade é o aspecto visual que se deseja controlar.
- O valor define como aquela propriedade será aplicada.

## Propriedades de Texto

### Color - define a cor do texto

```css
color: red;
color: #FF0000;
color: rgb(255, 0, 0);
```

### Font-size - define o tamanho da fonte
```css
font-size: 16xp;
font-size: 1.5em;
font-size: 1rem;
```

### Font-weight - define a espressura da fonte
```css
font-weight: normal;
font-weight: bold;
font-weight: 700;
```

### Font-style - define o estilo da fonte
```css
font-style: normal;
font-style: italic;
```

### Text-align - define o alinhamento horizontal do texto
```css
text-align: left;
text-align: center;
text-align: right;
text-align: justify;
```

### Text-decoration - define a decoração do texto
```css
text-decoration: none;
text-decoration: underline;
text-decoration: line-through;
```

### Line-height - define o espaçamento entre linhas
```css
line-height: 1.5;
line-height: 24px;
```

## Propriedades de Fundo

### Background-color - define a cor de fundo do elemento
```css
background-color: yellow;
background-color: #FFFF00;
background-color: transparent;
```

## Propriedades de Espaçamento

### Margin - define o espaço externo ao redor do elemento
```css
margin: 10xp;
margin: 10xp 20xp;
margin: 10xp 20xp 5xp 0;
```

### Padding - define o espaço interno entre o conteúdo e a borda
```css
padding: 8xp;
padding: 4xp 12xp;
```

## Propriedades de Borda

### Border - define a borda ao redor do elemento. Recebe largura, estilo e cor.

```css
border: 1px solid black;
border: 2px dashed red;
border: 3px dashed blue
```

### Border-radius - define o arredondamento dos cantos da borda

```css
border-radius: 4px;
border-radius: 50%;
```
